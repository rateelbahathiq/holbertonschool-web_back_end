# ES6 Basic

## 📚 Description

This project introduces the fundamental features of **ECMAScript 6 (ES6)**, the modern version of JavaScript used in Node.js and modern web development.

---
## 📖 Concepts Covered

- Using `const` and `let`
- Understanding block scope
- Writing arrow functions
- Using default parameters
- Working with rest parameters
- Using spread syntax
- Writing template literals
- Using object property shorthand
- Using computed property names
- Writing ES6 method properties
- Looping with `for...of`
- Creating and manipulating objects dynamically

---

## 📂 Files

| File | Description |
|------|-------------|
| `0-constants.js` | Uses `const` and `let` instead of `var` |
| `1-block-scoped.js` | Demonstrates block-scoped variables |
| `2-arrow.js` | Converts a standard function to an arrow function |
| `3-default-parameter.js` | Uses default function parameters |
| `4-rest-parameter.js` | Uses rest parameters to count arguments |
| `5-spread-operator.js` | Uses spread syntax to combine arrays and strings |
| `6-string-interpolation.js` | Uses template literals for string formatting |
| `7-getBudgetObject.js` | Uses object property shorthand |
| `8-getBudgetCurrentYear.js` | Uses computed property names |
| `9-getFullBudget.js` | Uses ES6 method properties |
| `10-loops.js` | Uses `for...of` loops |
| `11-createEmployeesObject.js` | Creates objects dynamically using computed properties |
| `12-createReportObject.js` | Creates a report object with methods |

## 💻 Example Usage
